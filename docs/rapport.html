Présentation

Le projet

IFRI MentorLink est une application web de mentorat académique développée dans le cadre du projet intégrateur PIL1 2025–2026. Elle met en relation les étudiants de l'IFRI pour du soutien académique et professionnel, en s'appuyant sur un algorithme de matching intelligent.

L'application permet à chaque étudiant de créer un profil détaillé (compétences, disponibilités, filière), de trouver des mentors compatibles, d'envoyer des demandes de mentorat et de communiquer en temps réel via une messagerie instantanée.

Fonctionnalités principales

  • ✓ Inscription et connexion sécurisées (hachage bcrypt, Flask-Login)
  • ✓ Profil utilisateur complet : bio, compétences, disponibilités horaires, filière, niveau
  • ✓ Algorithme de matching pondéré (compétences 50 pts, disponibilités 30 pts, filière 20 pts)
  • ✓ Système de demandes de mentorat avec suivi du statut
  • ✓ Messagerie instantanée en temps réel (Flask-SocketIO + Vue.js)
  • ✓ Tableau de bord personnalisé avec activité récente
  • ✓ Interface responsive (Tailwind CSS)

Intelligence

Algorithme de matching

Le score de compatibilité est calculé sur 100 points, répartis en trois composantes :

50

Compétences

Intersection entre les compétences du mentoré et celles du mentor, rapportée au nombre total de compétences du mentoré.

30

Disponibilités

Chevauchement temporel entre les créneaux horaires des deux utilisateurs, pondéré par la durée du créneau du mentoré.

20

Filière

Score fixe de 20 si même filière, 10 si filières différentes (toutes les filières sont proches à l'IFRI).

Formules

# Compétences (C sur 50)
C = (|compétences_mentor ∩ compétences_mentoré| / |compétences_mentoré|) × 50

# Disponibilités (H sur 30)  — meilleur créneau commun
H = (durée_chevauchement / durée_créneau_mentoré) × 30

# Filière (F sur 20)
F = 20  si même filière
F = 10  sinon

# Score total
Score = C + H + F   (sur 100)
Seuil bon match : Score ≥ 60
Instructions techniques

Déploiement

Cloner le dépôt GitHub

git clone https://github.com/Real-Win/PIL1_2526_34.git
cd PIL1_2526_34

Créer et activer l'environnement virtuel

# Linux / macOS
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate

Installer les dépendances

pip install -r requirements.txt

Dépendances principales : flask, flask-sqlalchemy, flask-login, flask-bcrypt, flask-socketio, pymysql.

Configurer la base de données MySQL

Créer la base de données et importer le schéma :

mysql -u root -p -e "CREATE DATABASE mentorlink CHARACTER SET utf8mb4;"
mysql -u root -p mentorlink < schema.sql

Puis modifier config.py :

SQLALCHEMY_DATABASE_URI = "mysql+pymysql://user:password@localhost/mentorlink"

Lancer l'application

python run.py

L'application est accessible sur http://127.0.0.1:5000.

Guide

Manuel d'utilisation

1. Créer un compte

Accéder à /inscription, remplir les champs nom, prénom, e-mail, téléphone, filière et rôle (mentor, mentoré ou les deux). Le mot de passe doit faire au moins 6 caractères.

2. Compléter son profil

Depuis /profil, renseigner ses compétences (séparées par des virgules) et ses créneaux de disponibilité. Ces informations sont essentielles pour que l'algorithme propose des correspondances pertinentes.

3. Trouver un mentor

Depuis /matching, l'algorithme analyse le profil et affiche les 3 mentors les plus compatibles avec leur score de compatibilité. Cliquer sur « Envoyer une demande » pour initier le mentorat.

4. Utiliser la messagerie

Depuis /messages, sélectionner un contact dans la colonne gauche pour ouvrir la conversation. Les messages s'affichent en temps réel grâce à Socket.IO. L'historique est conservé en base de données.

5. Tableau de bord

Le tableau de bord (/dashboard) présente un récapitulatif : messages non lus, complétion du profil, filière et date d'inscription. Il affiche aussi l'activité récente (derniers messages reçus).
