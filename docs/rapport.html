📘 IFRI_MentorLink - Rapport de projet

📌 Présentation

IFRI_MentorLink est une application web de mentorat académique permettant de connecter les étudiants selon leurs compétences, filières et disponibilités.

🎯 Objectif

Créer une plateforme permettant la mise en relation intelligente entre étudiants pour favoriser l'entraide académique et professionnelle.

🏗️ Architecture

  • Frontend : HTML / CSS / JavaScript
  • Backend : Python (Flask)
  • Base de données : MySQL / SQLite
  • Versioning : Git + GitHub
